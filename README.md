# bilibili-backup

备份那些“危险”的哔哩哔哩视频。

## 使用

在`links`当中贴`BVxxxxx`或`https://www.bilibili.com/video/BVxxxxx`，每行一个。

`yarn` 之后直接运行 `index.js` 或 `index.ts`。

导出后的格式均为 [哔哩哔哩 API](https://github.com/SocialSisterYi/bilibili-API-collect) 返回的数据格式，如有需要请自行转换。
